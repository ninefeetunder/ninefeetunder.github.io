css学习

发表于 2018-10-18 | 分类于 css | 阅读次数:

学习css也有一段时间了。准备写一些关于我在学习css过程中遇到的一些重要的知识点，总结一下自己学到的知识，这是第一篇。

理解css

css名为层叠样式表，顾名思义。用css给html写样式时是一层一层叠加上去的，最后形成一个样式被浏览器渲染出来。当然选择器的权重决定了最终的样式效果。

浏览器的默认样式

浏览器对html元素都有一个默认样式，当css没有对html元素进行样式设计时，浏览器就会让其的样式为默认样式。
注意这里的理解，浏览器并不是天生就认识那些标签，例如p标签是块级元素，并不是浏览器认识p标签，所以让它显示成块级元素。而是在浏览器的默认样式中为p标签设置了 

p {display: block;}

一句话，浏览器将载入的html变为dom树，但是此时没有任何显示样式。所以显示的样式，都是css定义的，浏览器只会通过css来渲染视图样式。说到底，还是css在控制html的样式。不同浏览器的默认样式都不相同，这也是在写css时在一开始会写上 

* {
margin: 0;
padding: 0;
}

的原因。

block元素和inline元素

block元素独占一行，可以设置width，height，margin等属性。
而对inline元素设置width，height属性时不会生效。但是当对inline元素设置float属性，position属性的值是absolute/fixed时，inline元素变成了”块”，就可以对其进行设置width，height属性了。

em和px

在设置文字大小或者和文字有关的距离时，如p的margin，line-height，font-size。可以用px或者em。px时绝对值，固定大小，是不可以改变的。当设置了样式为px是，其值就已经确定了。em是一个浏览器识别的长度单位，但它不是固定的，是相对的。默认情况下
1em = 16px ,这个值是可以更改的。通过设置 

body{
font-size: 20px;
}

此时，1em = 20px。所以当em被修改时，其他用em设置的样式也就自动进行了修改。这也是用em进行设置的好处。

inline-block

浏览器默认样式规定，button，textarea，input，object，select为inline-block样式。inline-block样式的特点是，能够设置width，height，margin等样式，又不会独占一整行。

最后再说一个css属性background-size。用于设置背景图片的大小。以及以下两种值的区
别 

background: 100% 100%;
background: 100%; /* 当只有一个值是另一个默认是auto*/

假设图片背景的长宽比是3:2。
第一种情况：图片要占据盒子的水平方向的100%，和垂直方向的100%。因此图片被拉伸变形，放入盒子中显示，看到是完整的图片，但图片已经变形。
第二种情况：图片占据盒子的宽度100%，但第二个值是auto。让图片自动根据占据盒子的宽度和图片的长宽比来拉伸图片，并放入盒子中。但是图片超出盒子高度的部分将会被截掉，看到的是不完整的图片，但图片并没有因为拉伸而导致变形。
