Json

发表于 2018-10-30 | 分类于 JavaScript

json是一种轻量级的数据格式，并不一种编程语言。

语法

json有三种类型的值：

  1. 简单值：这与JavaScript中的语法类似，可以再json中表示字符串，数值，null，布尔值，但是json不支持JavaScript中的特殊值undefined。

  2. 对象：对象是一种复杂数据类型，json中的对象是一组无序的键值对。每个键值对中的值可以是简单值，也可以是复杂数据类型的值。

  3. 数组：数组也是一种复杂数据类型，数组是一组有序的值得列表，可以通过数值索引访问其中的值。数组的值可以是任意的类型—简单值，复杂类型值。

需要注意的是在json中没有变量，函数，对象实例的概念。

简单值

简单值很简单，需要注意的就是json中的字符串必须加双引号。

解析与序列化

ECMAScript5对解析JSON的行为进行了规范，定义了全局对象JSON。JSON对象有两个方法：stringify()和parse()。在简单的情况下，这stringify()方法把JavaScript对象序列化为JSON字符串，parse()方法把JSON字符串解析为原生JavaScript的值。

值得注意的是经过序列化后的JSON字符串中，原JavaScript中的gf和fn属性都不见了。因为在序列化JavaScript对象时，所有函数及函数原型都会被有意的忽略，不体现在结果中，值为undefined的任何属性也都会被跳过。结果中最终都是值都是有效的JSON数据类型值。

JSON.parse()方法是把一个JSON字符串解析为JavaScript类型值。

如果传给JSON.parse()的字符串不是有效的json，该方法会抛出错误。
