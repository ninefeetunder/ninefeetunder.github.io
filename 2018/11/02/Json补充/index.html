JSON.stringify()该方法有三个参数。1.需要序列化的JavaScript对象。2.过滤器，可以是数组也可以是一个函数。3.一个选项，表示是否在JSON字符串中保留缩进。

可以看到返回的JSON字符串中只有选择器数组中存在的属性。

当选择器是函数时，这个函数接收两个参数，一个是键，一个是值。这个函数将作用于传入的JavaScript对象的每一个键值对上。

toJSON方法

先看一个例子

给对象定义了toJSON方法，该方法返回这个对象的name属性。然后再去序列化对象后就只得到了toJSON方法的返回值。这就涉及到了序列化对象的顺序了。

  1. 如果存在toJSON方法而且能通过它取得有效的值，则调用该方法。否则返回对象本身。
  2. 如果存在第二个参数则应用这个过滤器函数。传入函数过滤器的值是第一步返回的值。
  3. 对第二步返回的每个值进行相应的序列化。
  4. 如果提供了第三个参数，则执行相应的格式化。

JSON.parse()方法的第二个参数是一个函数，将在每个键值对上调用。这个函数被称作还原函数。如果还原函数返回undefined，则表示要从结果中删除相应的键，如果返回其他值，则将改值插入到结果中。
