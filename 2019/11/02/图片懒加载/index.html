图片懒加载

发表于 2019-11-02 | 分类于 性能优化

定义

图片的懒加载也叫延迟加载，在长网页中未出现在可视区的图片延迟加载，可以很好地优化网页的性能。用户滚动到它们之前，可视区域外的图像不会加载。

解决的问题

  • 提升网页加载速度
  • 防止无效的资源加载

实现思路

先将每个图片的src属性设置为占位图片，图片真实的url则设置在data-src属性上，
通过监听scroll事件判断 浏览器可视区高度 与当前 浏览器滚动条滚动的距离 之和是否大于 图片元素在文档中的距离，大于则说明该图片已经出现在浏览器的视口中（可能是当前视口中，也可能已经出现过了），则把图片的src替换为data-src的值，展示图片。不大于则说明带图片还未出现在浏览器的视口中，也就不急着去加载。

实现效果

没有出现在浏览器视口中的图片是有一个占位图片的，当我们滚动滚动条，下面的图片展示出来时，src属性才会被替换成真正需要展示的图片的url。也就实现了懒加载。
