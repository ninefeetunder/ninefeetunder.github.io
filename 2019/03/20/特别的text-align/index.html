你只是text-align啊

发表于 2019-03-20 | 分类于 CSS

昨天在写东西是遇到了这个一个样式

一个div元素被放到了水平居中的位置上，当然这个div被设置成了行内块元素。我以为是用定位的方法实现这种效果的，结果看了一下它的样式，发现只是普通的设置了宽高和padding，完全没有用到position。

后来发现是将这个div元素转换为行内块元素后，在其父元素上设置了text-align:center样式，就实现了这种效果。这个div元素就自动的移动到了中间。

text-align 属性规定元素中的文本的水平对齐方式。这是W3C对text-align用法的描述。只是对元素中的文本设置对齐的方式，但是现在这明显是对一个元素内的inline-block元素也起到了相同的作用。

经过测试，发现在块级元素中设置text-align: center;样式，则这个块级元素中的inline元素和inline-block元素都会自动水平居中。 这就说明设置text-align属性不仅仅只对元素内部的文本起作用，同时也会对这个元素中的inline元素，inline-block元素起作用。

令元素水平居中似乎多了一种方法~
