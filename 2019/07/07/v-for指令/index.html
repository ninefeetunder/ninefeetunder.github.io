v-for指令

发表于 2019-07-07 | 分类于 Vue

v-for是vue中的指令。

这个指令与其他指令行为有点不太一样，让我乍一看感觉有点蒙，所以记录一下。

v-for的行为总结为两句话。
v-for在遍历出数据时也会再次创建使用v-for指令的这个对象
v-for遍历出来的数据就只能在刚被创建出来的这个对象上使用

显示效果：

因为li使用了v-for指令从todos中遍历出todo，所以li标签也就被再次创建出来了。同时遍历出的每个todo也就只能用于同时被创建出的li中。

也就是说v-for指令可以同时渲染出多个元素，还要注意必须要给每个元素绑定key值。
