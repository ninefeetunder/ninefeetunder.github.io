em与rem的区别

发表于 2019-04-23 | 分类于 CSS

在css中常用的单位是px，em，rem。三者都是长度单位，但还是有区别的。

px是固定的像素，一旦确定就不可以再改变了。
em和rem是相对长度单位，用它们设置的长度都是通过某个参照计算出的长度。其中，
em是参照父元素。
rem的参照根元素(html元素)。

em

关于em还有一些规则：

  1. 子元素的字体大小是相对于父元素的font-size进行计算的。
  2. 元素的width/height/padding/margin/line-height用em，则是相对于元素自己的font-size进行计算的。

结果：

若在最外层没有指定font-size为具体的px，而是用em设置的话，则浏览器默认的 1em = 16px。

rem

rem的长度都是相对于html这个根元素的。一般的使用方法就是在html元素中设置font-size，然后所有rem长度都是相对于这个长度来进行计算的。

em，rem这部分的知识点比较简单，为了防止自己忘记，还是记录一下吧~
